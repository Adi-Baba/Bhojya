## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Bhojya.git`
3. Create a branch: `git checkout -b feature/your-feature-name`

## Development Setup

### Prerequisites
- Windows 10/11
- Clang/LLVM installed
- CMake 3.15+

### Building from Source
```bash
cd Bhojya
cmake -B build
cmake --build build --config Release
```

## How to Contribute

### Bug Reports
- Use GitHub Issues
- Include Bhojya code sample that reproduces the bug
- Include compiler version and OS

### Feature Requests
- Open an issue first to discuss
- Describe the use case
- Provide example syntax if applicable

### Code Contributions
1. Follow existing code style
2. Add tests for new features
3. Update documentation
4. Submit a Pull Request

## Code Style

### C++ (Compiler)
- Use camelCase for variables
- Use PascalCase for classes
- Comment complex logic

### Bhojya Language
- Sanskrit/Hindi keywords
- Use `+++` for comments
- Use `===` for block comments
